# ClickDose Labs Website

The marketing site for ClickDose Labs, a growth marketing agency. Hand-built, fast, and conversion-focused, with a shared design system across pages.

## What's here
- Homepage and service pages (Growth Marketing, Cold Email Lab)
- A personal profile page
- A shared CSS design system used across the subpages
- Calendar booking integration and full SEO/OG/favicon setup

## Tech
- Vanilla HTML, CSS, and JavaScript (no framework)
- Shared design tokens and components in `shared/styles.css`
- Deployed on Vercel
